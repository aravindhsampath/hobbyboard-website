# HobbyBoard Website

This repository contains the landing page, documentation, and asset files for the **HobbyBoard** project.

**Main Project:** [github.com/aravindhsampath/hobbyboard](https://github.com/aravindhsampath/hobbyboard)

## Structure

*   `index.html`: Main landing page with feature showcase.
*   `docs.html`: Comprehensive documentation.
*   `about.html`: Project philosophy and story.
*   `faq.html`: Common questions.
*   `contact.html`: Social links.
*   `css/`: Stylesheets.
*   `js/`: Frontend logic (theme, tabs, TOC).
*   `images/` & `fonts/`: Static assets.

## Local Development

To view the site locally, simply serve the directory:

```bash
python3 -m http.server 8000
# Open http://localhost:8000
```
